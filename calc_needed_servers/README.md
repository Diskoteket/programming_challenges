## Calculating needed servers
This is a programming-problem where you are given a set of input(via stdin) and need to calculate the amount of needed servers and output them to stdout.

### Sample input
4 1
0
1000
1000
1299

### Explanation of sample input
Entries can be in the range 0 - 100 000

* 4 = Amount of requests
* 1 = Amount of max requests per server, per millisecond
* 1000 = Entry, milisecond where a request happens
* 1000 = Entry, milisecond where a request happens
* 1299 = Entry, milisecond where a request happens
