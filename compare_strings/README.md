## Compare strings
This is a programming-problem where you are given a set of input(via stdin) and need to compare X strings and in a visual way present the differences.

### Sample input
3

ATCCGCTTAGAGGGATT
GTCCGTTTAGAAGGTTT
abcdefghijklmnopqrstuvwxyz
bcdefghijklmnopqrstuvwxyza
abcdefghijklmnopqrstuvwxyz0123456789
abcdefghijklmnopqrstuvwxyz0123456789

### Explanation of sample input

* 3 = Amount of incoming sets of strings to compare
* ATCCGCTTAGAGGGATT = First string, compare to Second string
* GTCCGTTTAGAAGGTTT = Second string
* etc....
